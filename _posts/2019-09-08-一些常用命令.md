# 常用命令



1. `sudo ntfsfix /dev/sdb4`	双系统挂载磁盘
2. `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple` pip设置源
3. Tmux使用：
   1. `tmux` :进入运行命令
   2. `crtl + b d`：退出当前会话
   3. `ctrl + b s`：列出所有会话 
   4. `ctrl + b x`：关闭当前窗格 
   5. `ctrl + b %`：划分左右两个窗格
   6. `ctrl + b "`：划分上下两个窗格 
   7. `ctrl + b arrow`：光标切换到其他窗格
